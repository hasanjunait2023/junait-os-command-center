# Junait OS

A founder operating system for Junait with realtime monitoring, integration registry, command center workflows, and execution routing through Blackforge.

## Purpose
Junait OS is the control layer above business operations, product execution, monitoring, automation, and decision support.

## Core components
- `master-architecture.md` — full operating system architecture
- `integration-registry.yaml` — all planned realtime integrations and status
- `monitoring-dashboard.html` — local command center dashboard
- `ops-playbook.md` — operating rules and execution flow
- `module-map.md` — A-to-Z module breakdown
- `launch-sequence.md` — phased rollout plan

## Design principles
- one command center
- one source of truth per module
- realtime status where possible
- founder-first visibility
- agent-assisted execution through Blackforge

This workspace is now initialized. It contains the architecture and control surface. Live integrations still need credentials, API hookups, and source connections.
